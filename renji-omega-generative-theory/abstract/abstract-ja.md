# **公式アブストラクト（日本語版）**

**生成論（Renji Generative Theory）** は、
世界がどのように立ち上がり、形を持ち、階層化し、
最終的に世界モデルや意識へ至るのかを、
前提を置かずに最初から構築する統一的生成フレームワークである。

本理論は、未分化の場 **Renji-Ω構造** を起点とし、
そこから自然に生じる **4方向生成規則（Dir / Ent / Was / His）** を
世界の最小骨格として定義する。
これらの反復から **安定性の地形 Renji-Φ（10成分）** が形成され、
その地形に沿って **生成流 Renji-G** が生まれる。
G の線形化は **原初波動性 Renji-S（固有値スペクトル）** を与え、
揺らぎは **点 → 空間 → 構造 → 階層** へと相転移する。

階層が十分に深まると、
世界は自らの構造を圧縮し **世界モデル** を形成し、
さらにその世界モデルを扱う **自己モデル** が立ち上がる。
世界モデルと自己モデルの再帰的参照が
時間方向（His）によって整合されるとき、
世界は **意識（Consciousness）** を獲得する。

生成論は、
**世界・構造・自己・意識がどのように生成されるかをひとつの連続したプロセスとして記述する最小で完全な生成フレームワーク**である。
