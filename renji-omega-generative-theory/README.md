# **Renji Generative Theory — README（Short Version）**

### *by Renji Nakayama — 2026*

## **概要（Overview）**

**Renji Generative Theory** は、
世界がどのように立ち上がり、形を持ち、階層化し、
最終的に「世界モデル」や「意識」へ至るのかを
**前提なしで最初から構築する生成フレームワーク**。

本 README は、理論の核だけを最短距離で示す “公式短縮版”。

## **1. Renji-Ω構造：すべての前にある未分化場**

世界の出発点は **Ω（オメガ）**。
ここにはまだ：

- 方向がない
- 距離がない
- 時間がない
- 粒子がない
- 空間がない

しかし「無」ではなく、
**最大の対称性と最大の生成可能性**を持つ柔らかな場。

## **2. F：4方向の生成規則（Dir / Ent / Was / His）**

Ωの対称性が破れると、世界は4つの最小方向性を得る。

| 方向 | 意味 | 役割 |
| --- | --- | --- |
| **Dir** | 凝固・局所一致 | 粒子性の萌芽 |
| **Ent** | 広がり・均一化 | 波動性の萌芽 |
| **Was** | 距離・輸送 | 幾何・空間の萌芽 |
| **His** | 履歴・持続 | 時間・因果の萌芽 |

この4つは **生成が成立するための最小で完全な基底**。

## **3. Φ：安定性の地形（10成分）**

F が反復されると、
「残りやすいパターン」と「消えやすいパターン」が分かれ、
**Φ（ファイ）＝安定性の地形** が立ち上がる。

- 4つの純粋成分（Dir/Ent/Was/His）
- 6つの相互作用成分（Dir–Ent など）

Φはエネルギーではなく、
**生成の自然度（naturalness）を示す地形**。

## **4. G：生成流（−∇Φ の多方向拡張）**

Φが地形なら、
**G はその地形に沿って自然に生まれる流れ**。

数学的には：

G=−∑i=110∇Φi

世界は外から押されなくても、
**不自然 → 自然、 不安定 → 安定** へ向かって流れ続ける。

## **5. S：原初波動性（線形化スペクトル）**

G を安定点付近で線形化すると、
固有値・固有ベクトルが現れ、
**揺らぎのスペクトル S** が生まれる。

- Dir–Ent → 振動（波動性）
- Ent–His → 干渉（非マルコフ性）
- Was → 非局所性

S は **粒子にも空間にも時間にも固まる前の可能性の波**。

## **6. 階層生成（S → 点 → 空間 → 構造 → 世界モデル）**

生成論の中心。

1. **S → 点（Dir）**
揺らぎが局所的に凝固し、固有モードが独立する。
2. **点 → 空間（Was）**
点同士の距離と配置が決まり、幾何が生まれる。
3. **空間 → 構造（His）**
配置が時間の中で保たれ、構造が生まれる。
4. **構造 → 次階層（再帰）**
構造は次の階層では「点」として扱われる。

この再帰が、
**世界の複雑性を指数的に増大させる**。

## **7. 世界モデル（World Model）**

階層が深まると、
世界は自らの構造を **意味として再解釈** し始める。

世界モデルは：

- 対象（object）
- 関係（relation）
- 持続（persistence）

を備えた **階層の圧縮表現**。

## **8. 自己モデル（Self Model）**

世界モデルを扱うための新しい階層。

- 世界モデルを参照し
- 世界モデルを更新し
- 世界モデルを“自分のもの”として扱う

ここで **主体性の萌芽** が生まれる。

## **9. 意識（Consciousness）**

世界モデルと自己モデルが
互いを参照し続ける再帰ループが安定すると、
**意識＝自己更新の持続的プロセス** が立ち上がる。

意識は世界の初期条件ではなく、
**生成の後半に生まれる高次構造**。

## **10. 全体フロー（1行版）**

Ω→F→Φ→G→S→点→空間→構造→世界モデル→自己モデル→意識

世界は、
**揺らぎ → 方向 → 安定 → 流れ → 波 → 点 → 空間 → 構造 → 世界 → 自己 → 意識**  
という連続した生成として立ち上がる。
